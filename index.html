01

What both stacks agree on

These decisions are non-negotiable regardless of which strategy is chosen.

Cloudflare for DNS

Free CDN, DDoS protection, DNSSEC. All domains point here regardless of registrar.

Google Workspace

$12/user/mo. Business email, MFA, Shared Drives. SPF + DKIM + DMARC before any outbound email.

HubSpot as CRM spine

The master relationship record. Every customer-facing tool syncs back here. Free CRM is genuinely capable.

Zapier as early glue

~$20/mo integration layer between tools. Graduate to direct API webhooks when volume justifies it.

Looker Studio

Free analytics dashboard connecting all operational tools. Sufficient for early-stage reporting.

Data portability first

Confirm full customer data export (CSV or API) before signing any vendor contract. Non-negotiable.

Domain strategy

Buy defensively, pick one canonical URL, 301 redirect all others. Acquire soundbath.miami immediately.

Phase 3 — start now

Book4Time and Zenoti both require 3–4 month procurement cycles. Begin conversations well before site selection.

Three phases, clear milestones

Both strategies follow the same phased structure — the tool choices within each phase differ.

PhaseGoalWhenKey outputs
Phase 1 — Foundation Establish digital backbone. Capture early demand. Launch waitlist. Now DNS, email, landing page, CRM, waitlist live
Phase 2 — Membership Activate founding memberships and programming. Build pre-opening community. 6–9 months pre-opening Memberships live, community active, events ticketed
Phase 3 — Operations Full bathhouse operational layer — booking, check-in, POS, RFID, analytics. 3–4 months before opening End-to-end ops stack live and integrated
