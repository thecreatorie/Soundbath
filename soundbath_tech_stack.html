DNS setup
Registrar strategy

Keep all domains at one registrar (Namecheap or Cloudflare Registrar). Then point all nameservers to Cloudflare DNS. Cloudflare Registrar charges at-cost — no markup.

Email authentication

Set up SPF, DKIM, and DMARC records in Cloudflare on day one. This protects deliverability and prevents email spoofing. Google Workspace walks through this — don't skip it.

Core principle: HubSpot is the spine. Every tool that touches a customer should connect back to HubSpot. This ensures that when you switch any single tool, you don't lose customer history. Avoid any platform that doesn't export your contact data.

Data flow — customer journey

Phase 1 → 3 data architecture

Framer (website) → Beehiiv (waitlist) → HubSpot CRM → Glofox (membership)
Luma (events) → Zapier → HubSpot CRM → Book4Time (bookings)
Book4Time + RFID → Looker Studio → Guest analytics
Long-term data ownership
Segment (later)

Once you have 3+ tools sending customer events, add Segment as a Customer Data Platform. All tools send events to Segment, which routes to HubSpot and any analytics tool.

Data portability rule

Before signing any contract: confirm you can export all customer data (name, email, visit history, spend) in CSV or via API. Any vendor who can't guarantee this is a risk.
